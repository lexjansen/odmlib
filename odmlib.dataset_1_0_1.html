class odmlib.dataset_1_0_1.model.ItemGroupData(**kwargs)[source]

Bases: ODMElement

One row (record) of data within a Dataset-XML dataset.

Corresponds to a single observation row in a tabular dataset. Contains one ItemData child for every variable in that row that has a value. The data:ItemGroupDataSeq attribute provides the row number for ordering records within the dataset.

class odmlib.dataset_1_0_1.model.ODM(**kwargs)[source]

Bases: ODMElement

The root element of a Dataset-XML 1.0.1 document.

A Dataset-XML file carries tabular clinical or reference data for submission. It references ODM metadata via StudyOID and MetaDataVersionOID and uses namespace http://www.cdisc.org/ns/Dataset-XML/v1.0 (prefix data:) for Dataset-XML-specific attributes such as ItemGroupDataSeq.
